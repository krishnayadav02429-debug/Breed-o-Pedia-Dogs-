🐶 Breed-o-Pedia (Dogs)
📌 Description

Breed-o-Pedia is a React app where users can explore different dog breeds, view images, and interact with a fun voting system.

🚀 Features
🔽 Select Breed → Sub-breed (Nested Dropdown)
🖼 View dog images in a gallery
🔄 Load more images
🎲 Random dog image generator
👍 Vote system (Hot 🔥 / Not ❌)
⏳ Loading indicators
🧠 Concepts Used
React Hooks (useState, useEffect)
API Fetching
Conditional Rendering
Component-based design
🌐 API

https://dog.ceo/dog-api/


🛠 Setup
git clone https://github.com/your-username/breed-o-pedia.git
cd breed-o-pedia
npm install
npm start
